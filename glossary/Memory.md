# Memory（記憶）

## 定義

Agent 跨會話保留的使用者個人化資訊，如偏好、習慣、歷史記錄。屬於使用者層級的長期儲存。

## 白話說法

就像你的助理有一本筆記本，記錄你喜歡靠窗座位、不吃牛肉、會員編號是多少。下次你找它，它不用重新問你。

## 與哪些概念容易混淆？

- **Knowledge Base**：記憶是針對個人的，知識庫是全體使用者共享的
- **Context**：記憶需要被載入 Context 才能使用，但兩者是不同的儲存層級
- **RAG**：RAG 是從知識庫檢索資訊的技術，記憶通常用更簡單的結構（如 JSON）儲存
