# Agent（智慧代理）

## 定義

能自主完成任務的 AI 系統。不是被動回答問題，而是主動規劃、執行、調整直到任務完成。

## 白話說法

Agent 像一個有手有腳的助理。你交代一件事，它自己想办法完成，過程中會自己決定下一步要做什麼。

## 與哪些概念容易混淆？

- **LLM**：LLM 是大腦，Agent 是完整的系統（大腦＋眼睛＋手腳）
- **Workflow**：Workflow 是固定步驟，Agent 動態決定步驟
- **Chatbot**：Chatbot 被動回答，Agent 主動完成任務
