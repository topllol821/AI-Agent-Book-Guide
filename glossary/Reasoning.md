# Reasoning（推理）

## 定義

LLM 使用邏輯步驟從已知資訊推導出結論的能力，是 Agent 決策的核心。

## 白話說法

推理就是「想」的過程。LLM 收到問題後，不是直接給答案，而是先思考：「這個問題是什麼意思？我需要哪些資訊？步驟是什麼？」

## 與哪些概念容易混淆？

- **Planning**：Planning 是拆任務，Reasoning 是推理出答案
- **ReAct**：ReAct = Reasoning + Acting，合起來才是完整的決策循環
- **Chain-of-Thought**：CoT 是 prompting 技巧，讓 LLM 逐步推理
