# Workflow（工作流程）

## 定義

固定的步驟序列，每一步由誰做、做什麼、做完後去哪裡都是預先定義好的。

## 白話說法

Workflow 就像自動化生產線——每個 station 做固定的事，順序不能亂改。跟 Agent 最大的差別是：Agent 可以自己決定下一步，Workflow 不行。

## 與哪些概念容易混淆？

- **Agent**：Agent 動態決定步驟，Workflow 固定步驟
- **Pipeline**：Pipeline 是資料處理流程，Workflow 是任務執行流程
- **Planning**：Planning 是 Agent 自己規劃步驟，Workflow 是工程師預先規劃
