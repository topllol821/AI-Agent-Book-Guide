# Harness（Agent 外圍工程／支撐系統）

## 定義

LLM 之外的所有工程設計，包括上下文的組織、工具的定義與安全、驗證機制、錯誤處理、回退策略。

## 白話說法

Harness 是「模型以外的工程」。就像馬車需要馬具（harness）才能讓馬拉得穩，LLM 也需要工程外殼才能可靠工作。

## 與哪些概念容易混淆？

- **Agent Framework**：Agent Framework 是 Harness 的一種實作，但 Harness 是更廣泛的設計理念
- **Prompt Engineering**：Prompt Engineering 只是 Harness 中的一小部分（Context 的設計）
- **RAG**：RAG 是 Harness 中的一種技術（知識檢索）
