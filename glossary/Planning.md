# Planning（規劃）

## 定義

Agent 將一個高層目標拆解為可執行的子任務序列，並動態調整的過程。

## 白話說法

就像你接到「辦一場派對」的任務，你會在腦中拆成：訂場地 → 邀請朋友 → 買食物 → 佈置。Agent 的 Planning 就是做這件事。

## 與哪些概念容易混淆？

- **Workflow**：Workflow 是工程師規劃的固定步驟，Planning 是 Agent 自己規劃
- **ReAct**：ReAct 是思考→行動的循環，Planning 是其中的思考環節
